# Java-experiments

#FilesAndNetwork - Программа для сбора данных из файлов с разным форматом: JSON, HTML, CSV
#ForkJoinPoolUsage - Программа с помощью ForkJoin создает карту сайта (карты из ссылок)
#Hibernate - Программа использует фреймворк Hibernate для подключения и внесения изменений в базу данных
#ImageResizer - Программа использует библиотеку imgscalr-lib для изменения размера изображений в многопоточном режиме
#SPBMetro - Программа реализует поиск кратчайшего пути по метро Санкт-Петербурга, а также реализовано модульное тестирование junit
#SQL - Программа реализует подключение и изменение данных в БД с помощью подключения к JDBC
#Transactions - Программа реализует имитацию перевода средств из разных банков в многопоточной среде. Также реализованы модульные тесты junit
